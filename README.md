# Automated Solar Tracker

## Overview

This project demonstrates the design and simulation of a single-axis automated solar tracking system. The tracker was designed in SolidWorks, the sun's movement was modeled using Python, and the motor response was simulated using MATLAB Simulink.

## Objectives

* Design a single-axis solar tracker.
* Calculate the required panel angle throughout the day.
* Simulate motor response using a first-order transfer function.
* Visualize the results using Python graphs.

## Software Used

* SolidWorks
* Python (NumPy, Matplotlib)
* MATLAB Simulink

## Project Structure

* `python/` – Python code and notebook
* `solidworks/` – CAD assembly and part files
* `simulink/` – Simulink model and scope output
* `report/` – Project report
* `images/` – Graphs and screenshots

## Results

The project successfully demonstrates the integration of CAD design, Python programming, and control system simulation for an automated solar tracking system.
